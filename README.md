# Symboli Renderer

> _Eclipse first, the rest nowhere._

Symboli Renderer improves the rendering of some applications.

## Build

Open `symboli_renderer.sln` and build it.
Of course you can use `msbuild` :

```bat
MSBuild -p:Configuration=Release
```
